class tables.Tables(score)¶

The Index class builds a variety of tables for convenient lookup

build()¶

Returns a dictionary of doubly nested lists. These include all of the data that can be obtained from the other methods in this class, stored as {‘clef’:{cleftable}, ‘key:{keytable}’, ‘time:{timetable}’}

Returns:
dictionary. A dictionary of the form described above
build_all_clefs()¶

Returns a list of lists of clef objects based on score passed to __init__ A list of clefs can be accessed for a given part via list[part_index]

Returns:
list
build_all_keys()¶

Returns a list of lists of keysignature objects based on all parts in the score. Access data for a given part with list[part_index]

Returns:
